Aula 07 - Operadores

Operadores Matemáticos

# + - operador de soma (e atribuição também).

# - - Operador de subtração.

# * - operador de multiplicação.

# / - operador de divisão.

# % - operador de módulo (resto da divisão.

# ** - operador de potência.

# += / -= / *= / /=... - operador de recebe o resultado da operação na frente por ele mesmo( se for f+=5 - recebe f + 5.)


É importante levar em consideração as regras de operações da matemática.


Operador de atribuição

# = - esse é o operador de atribuição.


Operadores de incremento e decremento

# variavel++ - a variável somando 1 no final.

# variavel-- a variável subtraindo 1 no final.

# ++variavel - a variável somando 1 no início.

# --variavel - a variavel subtraindo 1 no início.


Operadores relacionais

# == - igual.

# === - exatamente iguais, ou seja, mesmo tipo de variável e mesmo valor.

# < - menor que.

# > - maior que.

# <= - menor igual a.

# >= maior igual a.


Operadores lógicos

# ! - negação.

# || - ou (dijunção).

# && e (conjunção).


Operador ternário

Se algo for verdade ? acontece se for verdade : acontece se não for verdade.


A ordem de execução é sempre: Aritméticos (**, *, /, %, +, -), operadores relacionais(==, ===, <, >...), lógicos(!, &&, ||) e depois o ternário.
