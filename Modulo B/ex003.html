Aula 06

O + tem duas funções no JavaScript: a primeira é de concatenação, a segunda é um operador aritimético. Por isso, ao tentar somar dois número no prompt é necessário transformar a variável em um number. Veja o Exemplo abaixo:

var num1 = Number.parseFloat(window.prompt("Digite um número:"));

var num2 = Number.parseFloat(window.prompt("Digite outro número:"));

var soma = num1 + num2;

window.alert(`A soma dos números digitados é ${soma}.`);


Para tranformar uma variável string em número, utiliza-se o comando Number(variavel). Para transtormar em string utiliza-se String(variavel) ou vatiavel.toString().
